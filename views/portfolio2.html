Skills and Technologies

Front-End

Most of my projects contain HTML, CSS and ES6 JavaScript and the occasional jQuery. I have exposure to some popular front-end frameworks like Bootstrap and React, though there is much room for improvement with regards to their practical implementations. Dabbled in D3 as well for data visualisation in JavaScript.

Back-End

All my full-stack projects currently utilise NodeJS for server routing and MongoDB for database storage, with Python being used for AI-related projects. This website (as of November 2020) and all full-stack projects are currently hosted on Heroku, a cloud platform which I spent much of my time debugging when attempting deployment.

Others

As my preferred IDE, VS Code has been used for all projects done after my transition from Code Pen, during which I learnt the basics of Git and GitHub. I have used Microsoft Office software extensively in the past, and recently started to pick up Adobe Creative Cloud Software to aid in design.
